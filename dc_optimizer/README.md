# dc-optimizer

Библиотека для автоматизации планирования промышленных территорий и оценки параметров размещения центров обработки данных (ЦОД) в рамках научно-исследовательской работы (НИР).

## Возможности
- Проверка базовых инфраструктурных критериев участка под требования Tier III (площадь, подведённая электрическая мощность).
- Прогнозный расчёт климатической энергоэффективности (метрика PUE — Power Usage Effectiveness) на базе температурных режимов.

## Быстрый старт

```python
from dc_optimizer import verify_site_suitability, estimate_climatic_pue

# Проверка пригодности промышленной площадки (площадь в Га, мощность в МВт)
is_suitable = verify_site_suitability(area_hectares=3.5, power_megawatts=15.0)

# Оценка климатического PUE региона со средней температурой 12°C
pue = estimate_climatic_pue(average_temp=12.0)
